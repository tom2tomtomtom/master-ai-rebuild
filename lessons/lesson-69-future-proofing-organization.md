### Common Problems & Solutions

**Problem 1: "We're not sure which processes agents can actually automate"**

*Solution:* Use this decision tree:
```
Can an agent do this process?

1. Is it routine? (Yes/No)
   - If No: Agents can't do it (creativity needed)
   - If Yes: Continue

2. Are the rules clear? (Yes/No)
   - If No: Document rules first (e.g., "Approve if < $1K", "Escalate if angry")
   - If Yes: Continue

3. Does it touch multiple systems? (Yes/No)
   - If Yes: Is there an API? (might need integration work)
   - If No: Easier, agent can do it

4. Does it need human judgment? (Always/Sometimes/Never)
   - Always: Agents can't do it (keep human)
   - Sometimes: Agent handles routine, escalates judgment cases
```
