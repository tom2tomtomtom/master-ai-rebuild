```
1. Go to spline.design
2. Create new project

4. Add animation (rotate, scale, move)

```

### For Runway AI (Video to 3D)

```
1. Go to runwayml.com
2. Upload video or image
3. Use "3D Object Detection" or "Mesh Generation"
4. Export as 3D model
5. Import into Blender or Unity
6. Animate and render

Quality: 8/10, Time: 15 minutes
```
