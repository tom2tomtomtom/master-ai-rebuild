**Exercise 1: The Professional Image Prompt Formula**

**Scenario:** You need social media graphics for a product launch. You want consistency across 5 posts but each needs to highlight different features. In traditional workflows, this means briefing a designer, waiting for mockups, giving feedback, and paying for each iteration.

**The GPT-4o Image Prompt Framework:**

```

1. SUBJECT (What to show)
   - Main focus: [Primary subject]
   - Supporting elements: [Context/background]
   - What to avoid: [Things you don't want]

2. STYLE (How it should look)
   - Art style: [Photography/Illustration/3D/Minimalist/etc.]
   - Reference: "In the style of [Apple/Nike/Airbnb/specific aesthetic]"
   - Technical: [Lighting, angle, composition details]

3. COLORS (Palette)
   - Primary colors: [2-3 main colors with hex codes if specific]
   - Mood: [Warm/Cool/Vibrant/Muted/Monochrome]
   - Avoid: [Colors that conflict with brand]

4. TEXT (If needed)
   - Exact text: "[WHAT TO WRITE]"
   - Font style: [Bold/Modern/Elegant/Playful]
   - Placement: [Center/Top/Bottom/Corner]
   - Legibility: "Make text highly readable"

5. FORMAT (Technical specs)
   - Aspect ratio: [Square 1:1 / Landscape 16:9 / Portrait 9:16]
   - Use case: [Instagram post/LinkedIn banner/Presentation slide]
   - Resolution: "High quality for print/web"

6. MOOD/TONE (Emotional impact)
   - Feeling: [Professional/Playful/Serious/Innovative/etc.]

   - Target audience: [Who should this appeal to]

**Exercise 2: Mastering Iterative Design Refinement**

**Scenario:** You generated an image but it's 80% rightâ€”the colors are perfect, composition is great, but the text is too small and you want to add a logo area. In traditional workflows, this means another round with the designer. With GPT-4o, it's a 30-second iteration in the same conversation.

**Iterative Refinement Pattern:**

```

## ðŸ”§ Troubleshooting & Pro Tips

### Common Issues

**Problem 1: "Text in image is misspelled or garbled"**  
**Solution:** GPT-4o is much better than DALL-E 3, but for critical text, be explicit: "The text must be exactly: '[TEXT]' - no variations or misspellings." For very important text, consider adding it in post-production.

**Problem 2: "Image doesn't match my brand colors exactly"**  
**Solution:** Use hex codes: "Use exactly #1E3A8A for blue, #F97316 for orange" and say "Match these colors precisely, not approximate shades."

**Problem 3: "Style is inconsistent across images"**  
**Solution:** In same conversation thread, say "Generate next image in same style as previous one." Or save exact style prompt and reuse it.

**Problem 4: "Image looks AI-generated/fake"**  
**Solution:** Add "photorealistic," "professional photography," "high-end production" to prompts. Avoid over-prompting which can make images look artificial.

### Pro Tips

**ðŸ’¡ Technique 1: "Style Library"**  
Save prompts that work well: "Social_Template_Bold," "Presentation_Minimal," etc. Build reusable library of successful visual styles.

**ðŸ’¡ Technique 2: "Negative Prompts"**  
Tell GPT-4o what to avoid: "No cluttered backgrounds," "No cheesy stock photo vibes," "Avoid [specific thing]." This prevents common issues.

**ðŸ’¡ Technique 3: "Incremental Iteration"**  
Make one change at a time. Easier to get to perfect result than trying to fix multiple things at once.

**ðŸ’¡ Technique 4: "Brand Consistency Document"**  
Create doc with your brand colors (hex), preferred styles, mood descriptors. Reference it: "Use style from Brand Doc" to maintain consistency.

**ðŸ’¡ Technique 5: "Purpose-First Design"**  
Start with "This will be used for [specific purpose]" so GPT-4o optimizes for that use case (e.g., social media needs different composition than print).
