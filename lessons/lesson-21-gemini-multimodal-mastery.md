- Images (screenshots, photos, scans)
- Spreadsheets (paste as tables, or upload as images)
- Word/Google Docs (copy paste or upload)

```

- Invoice number

- Total amount across all invoices
- Consolidated line items (what were we buying from these vendors?)
```

**Step 5: Use Extracted Data**

### 5 Common Multimodal Problems & Solutions

**Problem 1: "Gemini misread information from images (bad OCR)"**

**Root Cause**: Image quality affects text extraction. Blurry images, poor lighting, small text, or complex layouts confuse Gemini's vision.

**Solution**:
1. Optimize image quality: High resolution (1200+ pixels wide), good lighting, straight angle
2. For documents: Upload PDF instead of image (more reliable)
3. For poor-quality images: Paste the text separately + include image for context
4. Verify critical data: Always spot-check extracted values against originals
5. Explicit instruction: "If you can't read text clearly, say so rather than guessing"

**Example fix**:
```
Instead of: [Upload blurry invoice image] "Extract invoice data"
Use: [Upload high-res invoice PDF OR clear image + paste invoice text]
     "Extract invoice data. If any field is unclear, flag it for manual verification."
```

---

**Problem 2: "Gemini gave me too much analysis; I only needed data extraction"**

**Root Cause**: Without clear constraints, Gemini provides comprehensive analysis. Sometimes you just need structured data.

**Solution**:
1. Specify output format: "Output ONLY: CSV table with these columns..."
2. Set length limit: "Structured data extraction only. Skip analysis/insights"
3. Use templates: Templates enforce focused output
4. Be explicit: "I don't need analysis, just data"

**Example fix**:
```
Instead of: "Analyze these documents"
Use: "Extract data ONLY into CSV format:
     Columns: Document Name | Date | Amount | Key Terms
     Skip: Analysis, insights, recommendations. Data extraction only."
```

---

**Problem 3: "The analysis missed important details I needed"**

**Root Cause**: You didn't specify what "important" means to you. Gemini's definition of important may differ from yours.

**Solution**:
1. Specify priorities: "Focus on: [3-5 specific things]"
2. Provide context: "This informs [specific decision]"
3. Give examples: "Look for issues like [examples of what matters]"
4. Ask follow-ups: "What did you find about [specific aspect]?"

**Example fix**:
```
Instead of: "Analyze these contracts"
Use: "Analyze these contracts focusing on:
     1. Liability caps (are they reasonable?)
     2. Termination terms (notice period, exit options)
     3. Payment schedule (is it reasonable for our cash flow?)
     4. Data handling (compliance with our privacy policy)
     5. Unusual terms (anything non-standard?)

     We're evaluating which contract is best for us, so prioritize these areas."
```

---

**Problem 4: "How do I ensure Gemini's analysis is accurate?"**

**Root Cause**: Multimodal analysis involves interpretation. Some subjectivity is inherent.

**Solution**:
1. Spot-check results: Verify 5-10% of analysis against originals
2. Use for first-pass only: Use Gemini to extract/analyze, human verifies critical findings
3. Compare consistency: If analyzing multiple items, check consistency (are similar items scored similarly?)
4. Request verbatim evidence: "For each conclusion, provide verbatim quotes/evidence"
5. Flag uncertainty: "If unsure about any finding, flag it for manual verification"

**Example fix**:
```
Instead of: [Upload 100 documents] "Extract all data"
Use: [Upload 100 documents + 5 verified examples]
     "Use these 5 examples as standard. Extract data from all 100 in same format/depth.
      After extraction, spot-check your work against the 5 examples.
      Flag any items where you're less than 90% confident."
```

---

**Problem 5: "I need to analyze videos but Gemini won't watch full videos"**

**Root Cause**: Gemini can't process video directly. It needs descriptions or transcripts.

**Solution**:
1. For short clips: Describe key moments or send transcript
2. For longer videos: Have someone take 5-10 screenshots at key moments, upload images
3. For transcripts: Paste transcript + description of what happens on screen
4. Hybrid approach: Combine transcript + screenshot images for best results
5. Manual first-pass: Have someone watch, create detailed outline, Gemini analyzes outline + key moments

**Example fix**:
```
Instead of: [Try to upload video] "Analyze this video"
Use: Transcript of video [paste text] + description of key moments
     Screenshots from [timestamp] showing [what happens]
     "Based on this transcript and these key moments, provide analysis of..."
```
