## üîß Troubleshooting & Pro Tips

### Common Problems & Solutions

**Problem 1: Voice Sounds Robotic or Unnatural**

Symptoms:
- Voiceover sounds artificial or mechanical
- Phrasing sounds unnatural
- Missing emotional inflection
- Rhythmic cadence sounds off

Causes:
- Poor script (unnatural phrasing)
- Wrong voice chosen for script tone
- Too-fast speech rate
- Too-slow speech rate

Solutions:
1. Rewrite script for natural speech:
   - Before: "Utilize this platform for operational optimization"
   - After: "Use this platform to work smarter, faster, better"
2. Choose different voice:
   - Test 3-5 voices with same script
   - Find voice matching script tone
3. Adjust speech settings:
   - Reduce speed: 0.85-0.95x for conversational feel
   - Increase clarity: Sometimes helps with delivery
4. Break up long sentences:
   - Before: "The comprehensive platform provides seamless integration, advanced analytics, and robust security features"
   - After: "This platform is comprehensive. It provides seamless integration. Advanced analytics. Robust security."

Example Fix:
```
Before (Robotic):
"Welcome to our innovative solution that revolutionizes your workflow
through advanced technological integration and data-driven insights."

After (Natural):
"Welcome to our solution. It revolutionizes how you work.
With advanced technology and real data insights. Let me show you how."
```

**Problem 2: Generated Music Doesn't Match Video Tone**

Symptoms:
- Music feels wrong for content
- Upbeat music for serious content (or vice versa)
- Wrong tempo for video pacing
- Instrumentation doesn't match brand

Causes:
- Vague prompt description
- Wrong mood/style selected
- Music generation interpreted prompt differently

Solutions:
1. Be more specific in prompt:
   - Before: "Professional music"
   - After: "Professional corporate background music. Serious, confident tone. Modern but not trendy. 60-70 BPM. Orchestral with electronic elements. 60 seconds length."
2. Specify mood explicitly:
   - Serious/corporate: "Powerful, confident, professional"
   - Creative/startup: "Energetic, innovative, modern"
   - Luxury: "Elegant, sophisticated, refined"
   - Educational: "Clear, straightforward, engaging"
3. Describe instrumentation:
   - Modern: "Electronic, synthesizers, modern production"
   - Professional: "Orchestral strings, piano, subtle"
   - Energetic: "Electric guitar, drums, upbeat"
4. Generate 5 variations:
   - Run same prompt 5 times
   - Select the best variation
   - Use others as backups

Symptoms:
- [CUSTOMER_NAME] appears in audio instead of actual name
- Variables showing as literal text
- Automation failed to substitute variables

Solutions:

2. Test with single customer first:
   - Don't automate yet

3. Check CSV formatting:
   - No extra spaces around names
   - All rows have all fields
   - Use UTF-8 encoding
4. Test automation with 5 customers:

   - Check file naming is correct

**Problem 4: Audio Quality Too Compressed or Distorted**

Symptoms:
- Audio sounds compressed or squashed
- Distortion or clipping in loud parts
- Quality loss when downloaded
- Sounds "small" or low quality

Causes:
- Wrong file format selected
- Export settings too compressed
- Platform applied too much processing
- Audio levels too high during generation

Solutions:
1. Download in highest quality format:
   - ElevenLabs: Select "MP3 (highest quality)" if available
   - Suno: Download as WAV (uncompressed) if available, MP3 if not
   - Descript: Export as high-quality MP3 (320 kbps)
2. Check audio levels:
   - Should peak at -6dB to -3dB, not 0dB (distortion)
   - If clipping, regenerate with lower clarity setting
3. Use uncompressed formats:
   - For editing: Download as WAV or AIFF
   - For distribution: MP3 at 256-320 kbps minimum
4. Post-process if needed:
   - Light compression: -2dB
   - EQ: Slight boost 2-4kHz for clarity
   - Normalization to -3dB peak
